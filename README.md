# API INMUEBLES FINCA

## Métodos disponibles

-Manipulación de datos de inmuebles

Obtener todos los inmuebles
GET /api/inmuebles/

Obtener inmueble por Id
GET /api/inmuebles/:InmuebleId

Crear un inmueble
POST /api/inmuebles/

Modificar un inmueble
PUT  /api/inmuebles/:InmuebleId

Borrar un inmueble
DELETE /api/inmuebles/:InmuebleId
